# Debris Flow Dam Simulator

* これは2026-07-15 22:49（Asia/Tokyo）にCodexが初期構築したプロジェクトである。
* KANAko 1.44の公開資料と実機挙動を参照し、クリーンルームで土石流・砂防ダム統合シミュレーターを構築する。
* 配布成果物は、主要機能がオフラインで動く単一HTML `debris_flow_dam_simulator.html` とする。
