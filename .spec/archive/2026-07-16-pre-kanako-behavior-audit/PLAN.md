# PLAN - 土石流・砂防ダム統合シミュレーター

## やりたいこと

- 2011年のKANAko 1.44を操作・表示の参考にしつつ、公開論文に基づくクリーンルーム実装で現代的なシミュレーターを作る。
- 高精度1Dを確立し、その同じ単一HTMLへ1D–2D結合、高度二相流、感度・不確実性分析まで段階的に統合する。
- 見た目のリアルさだけでなく、保存則、濡れ乾き、正値性、収束性、再現性、校正可能性を重視する。
- コンポーネント化して保守性を確保しながら、外部依存なし・軽量・オフライン中心にする。
- CSVや外部タイルを含むクライアント入力を信用せず、正規化層とWorker境界で二重に検証する。

## 完成像

- `debris_flow_dam_simulator.html`を開くだけで、参照ケースの編集、計算、可視化、自己検証、CSV入出力ができる。
- 標準1D、結合1D–2D、高度二相流（実験）を同じUIから切り替えられる。
- 水・運動量・粒径別土砂・河床の収支誤差を常時確認できる。
- 失敗理由、入力エラー、計算量超過、外部地形取得失敗を明示し、黙って補正しない。
