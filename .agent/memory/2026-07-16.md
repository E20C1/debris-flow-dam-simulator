# MEMORY

## プロジェクト概要

- 単一HTMLで動く土石流・砂防ダム統合シミュレーター。
- 標準1D、多粒径河床変動、複数砂防ダム、保存的1D–2D結合、高度二相流、感度・不確実性分析を段階統合する。
- 配布用ファイル入出力はCSVのみ。主要計算はオフラインで動作し、国土地理院DEMは任意のオンライン補助とする。

## 恒久的な制約

- 実装範囲と合格基準は `.spec/SPEC.md`、進捗は `.spec/TODO.md` を正とする。
- KANAkoは公開仕様・公開論文・ブラックボックス出力比較だけを参照し、コードを流用しない。
- すべての外部入力とWorker境界を検証し、検証前の値を計算カーネルへ渡さない。
- 数値モデルの差を特殊処理で隠さず、保存誤差と差分理由を表示する。
